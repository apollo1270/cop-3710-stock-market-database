# COP 3710 Project: Stock Market Time-Series Analytics Database

This project designs a relational database for storing historical stock market time-series data (prices and volumes) and supporting analytics queries such as moving averages and volatility. The database is designed for performance using date-based organization and indexing strategies for fast time-range queries.

**User Groups:** Financial analysts, data science students, and investment firms.
**Data Source:** Kaggle Stock Market Dataset (jacksoncrow).
