# Project Title: Stock Market Time-Series Analytics Database

## Application Domain
This project is in the financial data analytics and database systems domain. The goal is to build a database that stores historical stock prices and supports time-series analysis.

## High-Level Goals (Tentative Scope)
The goal of this project is to store historical stock price data in a database and perform analytical queries such as moving averages and volatility analysis. The database will use date-based partitions and indexing strategies to optimize query performance.

## Target Users
This project is intended for financial analysts and data science students who need structured access to historical stock market data for analysis and learning database optimization techniques.

## Data Sources
The primary data source is the Stock Market Dataset from Kaggle:
https://www.kaggle.com/datasets/jacksoncrow/stock-market-dataset

This dataset contains historical daily stock prices and trading volumes for multiple publicly traded companies.
